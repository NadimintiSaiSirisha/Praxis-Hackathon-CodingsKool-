Merge Sort

Like QuickSort, Merge Sort is a Divide and Conquer algorithm. It divides input array in two halves,

calls itself for the two halves and then merges the two sorted halves. The merge() function is used

for merging two halves.

INPUT:

First line of input contains one integer indicating the length of the array to be sorted. The second line of input contains the elements of the array, separated by space.
